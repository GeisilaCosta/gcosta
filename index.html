Monitore

monitore é um sistema web que fornece previsões meteorológicas e informações sobre o nível de água para ajudar comunidades a se prepararem para desastres naturais. Utiliza HTML, CSS, JavaScript, a API OpenWeatherMap e Leaflet.js para fornecer dados meteorológicos e mapas interativos

Visão Geral O sistema exibe dados meteorológicos e informações de risco de enchentes em tempo real através de um mapa interativo, permitindo que os usuários visualizem regiões de risco e detalhes climáticos. Funcionalidades Dados Meteorológicos em Tempo Real: Informações atualizadas sobre o clima, como temperatura, umidade e velocidade do vento. Alertas de Enchentes: Notificações sobre condições que podem levar a enchentes. Mapa Interativo: Visualização de áreas de risco e detalhes geográficos. Visualização de Níveis de Água: Dados sobre níveis de água e alertas em locais propensos a enchentes. Tecnologias Utilizadas HTML, CSS e JavaScript: Para estrutura, estilo e interatividade. OpenWeatherMap API: Para dados meteorológicos em tempo real. Leaflet.js: Para mapas interativos. Futuras Implementações Notificações baseadas em geolocalização. Camadas adicionais de mapa para previsões detalhadas e dados históricos. Integração com redes sociais para compartilhamento de alertas

Gam3r Store

Este é um projeto de e-commerce desenvolvido com tecnologias modernas e eficientes para criar uma plataforma robusta e escalável. O objetivo é fornecer uma interface rápida e intuitiva para usuários, enquanto o backend gerencia dados de forma segura e eficiente

Tecnologias Utilizadas
React: Para a construção da interface do usuário, oferecendo uma experiência interativa e responsiva.
Next.js: Framework React utilizado para renderização do lado do servidor (SSR) e geração de páginas estáticas, otimizando o SEO e o desempenho.
NestJS: Framework de backend Node.js eficiente para construir uma API robusta e escalável com TypeScript.
Tailwind CSS: Biblioteca de CSS utility-first para estilização rápida e customizável.
Prisma: ORM para gerenciamento de banco de dados, simplificando as consultas e mantendo a consistência dos dados.
PostgreSQL: Banco de dados relacional para armazenamento seguro e eficiente das informações de produtos, usuários e pedidos.

Vida Segura

Conscientização sobre Enchentes no Rio Grande do Sul 🌊 Objetivo: O projeto visa educar e preparar as comunidades do Rio Grande do Sul para enfrentar enchentes, fornecendo informações claras e acessíveis sobre como agir antes, durante e depois desses eventos, minimizando riscos e danos.

Problema: Muitas comunidades não têm acesso a informações e orientações adequadas para se preparar e lidar com enchentes, resultando em grandes prejuízos e riscos à vida. Funcionalidades: Dicas de Prevenção: Ações práticas para minimizar riscos antes de uma enchente. Instruções de Segurança: Passos detalhados para garantir segurança durante e após uma enchente. Mapas das Áreas de Risco: Visualização das regiões mais vulneráveis no RS, categorizadas por níveis de risco. Recursos e Contatos Locais: Lista de contatos importantes para facilitar o acesso a ajuda. Links Úteis e Fórum: Recursos adicionais e espaço para interação entre usuários. Tecnologias Utilizadas: HTML e CSS: Para uma interface leve, acessível e responsiva. Imagens e Mapas: Fornecem uma visão clara das áreas de risco sem sobrecarregar a página. Estrutura do Projeto: /pages: Páginas principais da plataforma. /img: Imagens usadas no site. styles.css: Arquivo de estilos para design consistente. Possíveis Melhorias Futuras: Área de Relato de Experiências: Para compartilhamento de relatos e dicas pessoais. Integração com Redes Sociais: Para expandir o alcance e aumentar a conscientização. Impacto Social: A plataforma busca capacitar e educar as comunidades, contribuindo para a segurança e resiliência das pessoas em áreas vulneráveis.

Influencers4You

Influence4You é uma plataforma de marketing de influenciadores que conecta empresas e influenciadores para colaborações em campanhas. Com o Influence4You.

As empresas podem encontrar influenciadores relevantes para promover seus produtos e serviços, enquanto os influenciadores podem descobrir oportunidades de parceria. Tecnologias Utilizadas Java: Usamos Java para o backend e a lógica de negócios. TypeScript: O frontend é construído com TypeScript e React. CSS: Estilizamos a interface com CSS e componentes reutilizáveis. Funcionalidades Principais Pesquisa de Influenciadores: Os usuários podem pesquisar influenciadores com base em nichos, seguidores, localização e outras métricas. Os influenciadores têm perfis detalhados com informações sobre suas áreas de interesse, público-alvo e taxas de colaboração. Gestão de Projetos: As empresas podem criar projetos e convidar influenciadores para colaborar. Os influenciadores podem aceitar ou recusar convites e acompanhar o progresso dos projetos. Monitoramento de Campanhas: As empresas podem rastrear o desempenho das campanhas, incluindo métricas como alcance, engajamento e conversões. Os influenciadores podem ver o desempenho de suas postagens e receber feedback das empresas.

BarberShop

Descrição Geral: O Projeto FSW Barber é uma aplicação web desenvolvida com Next.js e React, destinada a ser uma plataforma para barbearias. A aplicação inclui uma página que lista as barbearias cadastradas com base em parâmetros de busca.

Componentes Principais: Header: Renderiza o logotipo da aplicação e um botão de menu lateral que ativa o componente SidebarSheet. Search: Permite ao usuário buscar barbearias por título ou serviço. BarbershopItem: Renderiza informações e serviços de uma barbearia individual. BarbershopsPage: Página principal que lista as barbearias cadastradas com base em parâmetros de busca. Funcionalidades: Listagem de barbearias com base em parâmetros de busca (título ou serviço). Componente de busca para encontrar barbearias específicas. Cabeçalho com logotipo e menu lateral. Tecnologias Utilizadas: Next.js: Framework para desenvolvimento de aplicações web com React. React: Biblioteca JavaScript para construção de interfaces de usuário. Prisma: ORM para interação com o banco de dados. TypeScript: Linguagem de programação para desenvolvimento web. Banco de Dados: Utiliza Prisma para interagir com o banco de dados, armazenando informações das barbearias, como nomes e serviços oferecidos. Arquitetura: Baseada em componentes React, cada um responsável por uma funcionalidade específica. A página principal é o BarbershopsPage, com o Header gerenciando o cabeçalho e o Search facilitando a busca de barbearias.
